## 1. Lời mở đầu

Nhân một buổi chiều trời âm u tối hù, còn 1 tiếng nữa phải lao ra ngoài đường dòng xe hối hả để về nhà thì mình quyết định ngồi học Backend sau nhiều năm ngồi canh CSS, Flexbox, Column, Color,... các kiểu căng mắt.

Đọc đến đây mình nghĩ các bạn sẽ thắc mắc **Ủa gì dzậy trời sắp mưa thì liên quan gì đến học Backend**, lấy ngữ cảnh hoa mỹ vậy thôi chứ mình cũng "me học" lâu nay rồi, để không trì hoãn việc đó nữa nên mình tận dụng một hôm "rảnh rỗi" để làm thôi.

Trước đó mình có tham khảo nhiều nguồn, nhiều người quen rồi cho nên mình quyết định bắt đầu về việc học để biết **Query, Table là gì, how to Drop Database làm sao** cho nên hà há `MySQL - You are the chosen one`.

## 3. "Múa" thử vài câu lệnh MySQL nhé

Trong trường hợp bạn đang ngồi làm việc ở quán cafe, hay văn phòng của bạn có nhiều bạn nữ thì làm gì cho ngầu. So easy mình bật Terminal vào và gõ vài đường quyền chơi. Tuy nhiên, để **tương tác với MySQL trên Terminal** các bạn cần cài đặt PATH cho nó.

<CenterImageInBlog src="/static/images/Blog/path-find-zshrc.png" alt="Path to find zshrc file" />

Máy của mình thì sử dụng file `.zshrc` tuy nhiên nếu bạn dùng file `.bash_profile` thì cũng không thành vấn đề nhé. Bạn có thể mở file đó nhanh bằng cách bật Terminal lên và gõ

```bash
 open -t .zshrc
```

Kế tiếp bạn gõ trỏ PATH MySQL vào theo dòng lệnh mình để sẵn bên dưới nhé và bạn **Save file**.

```bash
 export PATH="$PATH:/usr/local/mysql/bin"
```

<CenterImageInBlog src="/static/images/Blog/config-mysql-path.png" alt="Config MySQL Path" />

Bạn có thể kiểm tra lại cài đặt đúng hay chưa bằng câu lệnh sau:

```bash
 mysql -u root -p
```

Nếu như mọi thứ ok thì Terminal sẽ bắt chúng ta đăng nhập mật khẩu cho `root user`. Mọi người dùng mật khẩu mình đã điền phía trên điền lại nhé. Nếu như mọi thứ diễn ra tốt đẹp thì Terminal của bạn lúc này sẽ xuất hiện màn hình welcome như ảnh dưới.

<CenterImageInBlog
  src="/static/images/Blog/interact-mysql-on-terminal.png"
  alt="interact MySQL on terminal"
/>

Lúc này bạn có thể gõ vài câu lệnh rồi nhấn **Enter** mà mình mới xem được trên mạng

```bash
 show databases;
```

Nếu bạn muốn thoát ra, bạn chỉ cần gõ `exit`.

## 4. Cài đặt Database Management

Và công cụ cuối cùng mình cần cài đặt đó chính là **Database Management**, vì là người mới mình cũng chưa biết dùng tool nào ok nên mình sẽ để hai option bên dưới cho mọi người chọn lựa nha, cá nhân mình thì gài cả hai luôn để xem thử dùng cái nào tiện hơn.

- [Table plus](https://tableplus.com/download)
- [MySQL Workbench](https://dev.mysql.com/downloads/workbench/)

Viết đến đây thì cũng hơn một tiếng rồi, cảm ơn bạn đã xem đến đây và hẹn gặp lại bạn trong bài viết tiếp theo nhé! 🙌
